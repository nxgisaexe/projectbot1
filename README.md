Back then, this bot had the following features:
## Features
- Made with JavaScript, Node.js, and Discord.js.
- Contains 80+ commands spanning moderation, utility, menu selection, music, reminders, reaction roles, and chatbot features.
- Further details on commands created on **minagichan**:
    - Moderation features for a discord server, including the base features to Warn, Mute, Kick, and Ban server members.
    - Avatar and emoji image commands to obtain direct image files from a user’s profile photo and custom emojis.
    - Timed reminders can be set up with the bot, that will DM you with your reminder after the desired time has passed
    - Music playback on Voice Call using ytdl-core and ffmpeg, supporting keyword search as well as direct media links from YouTube and Spotify. Included play/stop/pause/skip/queuing song commands. Made prior to discord’s limitations on music bot support. Since then, there are new ways to get a music bot working regardless of these limitations, which I later implemented on my newer discord bots.
    - Study session hosting command that sends time-sensitive embed notifications and role mentions to notify community members to join Voice Call.
    - A `,say [text]` command for the bot to repeat user-provided text
    - Food menu system containing 50+ F&B options which sent themed GIF embeds, as well as an 8ball roll system to show a random result to a question
    - Reaction role functionality for users to get roles by reacting to a message or by doing a specific command as long as prerequisite criteria have been met. For example, a user can only get a certain role after getting a prerequisite role.
    - Experimental support for cross-bot interactions where **minagichan** could communicate with another Discord.js bot through pre-programmed triggers.
    - Maintained **minagichan** for a ~500 member server before the server later closed

### My Takeaways
- Learned JavaScript, Node.js, and Discord.js through this project
- Learned how software changes over time, as Discord.js moved from prefixes to slash commands
- Improved my understanding in making practical tools for studying and entertainment purposes, while taking into account several use cases for discord bots that could be created
